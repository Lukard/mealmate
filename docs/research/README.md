## Adding Research Documents

When adding research findings:

1. Use descriptive file names: `YYYY-MM-DD-topic-name.md`
2. Include date, author, and methodology
3. Summarize key findings at the top
4. Link to raw data if applicable
5. Note actionable recommendations

## Template

```markdown
# [Research Title]

**Date:** YYYY-MM-DD
**Author:** Name
**Type:** Market/Technical/User Research

## Summary
Brief overview of findings

## Methodology
How the research was conducted

## Findings
Detailed findings

## Recommendations
Actionable next steps

## References
Sources and links
```
