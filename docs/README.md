## Project Overview

Meal Automation is a comprehensive meal planning and grocery automation system designed specifically for the Spanish market. It integrates with major Spanish supermarkets to provide:

1. **Intelligent Meal Planning** - Personalized meal suggestions based on dietary preferences, family size, and budget
2. **Supermarket Product Matching** - Automatic matching of recipe ingredients to supermarket products
3. **Price Comparison** - Compare prices across multiple Spanish supermarkets
4. **Automated Shopping** - Browser extension for one-click grocery purchasing

## Technology Stack

- **Frontend**: React + TypeScript
- **Backend**: Node.js + TypeScript
- **Scraping**: Playwright
- **Extension**: Chrome Extension API (Manifest V3)
- **Database**: PostgreSQL (planned)
